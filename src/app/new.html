Supply Chain & Warehouse Management System Use the same style keywords for every prompt so the design stays consistent. Global Style Instruction (add to every prompt) Style: modern SaaS dashboard UI, minimalistic, professional enterprise design, soft shadows, rounded corners, blue and white color palette, clean typography, responsive layout, smooth hover animations. 1️⃣ Welcome / Landing Page Prompt Design a modern welcome page for a Supply Chain & Warehouse Management System web application. The page should introduce the platform and guide users to login. Include: Hero section with headline “Smart Supply Chain & Warehouse Management” Short description about managing inventory, vendors, and logistics Illustration related to warehouse operations CTA button Login Secondary button Learn More Clean footer with About, Contact, Privacy Policy Style: modern SaaS landing page, minimalistic, professional, blue theme. 2️⃣ Login Page Prompt Design a modern login page for a Supply Chain & Warehouse Management System. Include: Company logo Login form with Email/Username and Password Login button Forgot Password link Optional Remember Me checkbox Subtle warehouse themed background illustration Style consistent with the welcome page. 3️⃣ Admin Dashboard Prompt Design a dynamic admin dashboard for a Supply Chain & Warehouse Management System. Layout: Top navigation bar with notifications, settings, and user profile dropdown Collapsible sidebar navigation Dashboard widgets: Total Inventory Pending Purchase Orders Shipments in Transit Low Stock Alerts Include: Line chart showing inventory trends Pie chart showing warehouse space usage Recent orders table Quick action buttons for Add Inventory, Create Purchase Order, Generate Report 4️⃣ Inventory Management Page Prompt Design an inventory management page for a warehouse management system. Include: Search bar Filter by category, warehouse, stock status Inventory table with columns: Item Name, SKU, Warehouse, Quantity, Reorder Level, Status Features: Add new inventory button Edit and delete actions Low stock highlighted in red Include inventory summary cards at the top. 5️⃣ Warehouse Management Page Prompt Design a warehouse management interface showing multiple warehouse locations. Include: Warehouse cards showing capacity and current stock Visual storage utilization bar Warehouse list table Add warehouse button Warehouse location map view (optional) 6️⃣ Vendor Management Page Prompt Design a vendor management page for a supply chain system. Include: Vendor list table Columns: Vendor Name, Contact, Location, Rating, Status Search and filter options Add vendor button Vendor performance rating indicator 7️⃣ Purchase Requisition Page Prompt Design a purchase requisition page where warehouse managers can request items. Include: Form to select item Quantity input Warehouse location Reason for request Submit requisition button Show table of submitted requisitions with status: Pending, Approved, Rejected. 8️⃣ Purchase Order Page Prompt Design a purchase order management page for procurement. Include: Purchase order list table Columns: PO Number, Vendor, Items, Quantity, Status, Delivery Date Features: Create new purchase order Approve or reject order Track delivery status 9️⃣ Goods Receipt Note (GRN) Page Prompt Design a goods receipt interface for warehouse staff. Include: PO number selection Item verification list Quantity received input Damage report checkbox Submit GRN button 🔟 Order & Shipment Tracking Page Prompt Design a shipment tracking dashboard. Include: Shipment list table Status badges: Pending, Shipped, In Transit, Delivered Shipment timeline progress bar Map showing shipment location 1️⃣1️⃣ Analytics & Reports Page Prompt Design an analytics dashboard for supply chain performance. Include charts: Inventory turnover Vendor performance Order fulfillment rate Warehouse utilization Include downloadable reports. 1️⃣2️⃣ Notifications & Alerts Page Prompt Design a notification center for warehouse alerts. Include: Profile settings Notification preferences System configuration Warehouse configuration 🧠 Pro Tip (Important for Your Project) Your actual module flow should be: Login ↓ Dashboard ↓ Inventory ↓ Warehouse ↓ Vendors ↓ Purchase Requisition ↓ Purchase Orders ↓ GRN ↓ Stock Update ↓ Orders / Shipments ↓ Analytics & Reports If your UI follows this workflow, your project will look like a real enterprise SCM system, not just a basic dashboard.
